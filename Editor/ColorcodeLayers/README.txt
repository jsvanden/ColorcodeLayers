COLORCODE LAYERS

:DESCRIPTION:
	This plugin visually displays what layer each object is in.

:NOTE:
	This only colorcodes objects which have 3D meshs attached to them.
	
:CONTENTS:
	README.txt
	AlterLayerColors.cs
	ColorData.cs
	DrawLayerColors.cs
	ToggleLayerColors.cs

:HOW TO USE:
	1) Ensure that Gizmos are enabled
	2) Use "Tools" > "Colorcode Layers" > "Toggle Layer Colors" or CTRL-G to toggle colorcoded layers.
	3) Use "Tools" > "Colorcode Layers" > "Alter Layer Colors" or CTRL-SHIFT-G to edit plugin settings.
